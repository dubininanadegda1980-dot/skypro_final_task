Git

  • git status - показывает текущее состояние репозитория, помогает увидеть/понять какие изменения сделаны, ещё закоммичены и не отслеживаемость файлов
  • git add - добавляет изменения в индекс, перед тем,как закоммитить
  • git commit - сохраняет изменения (локально)для отправки в удалённый репозиторий
  • git push origin main - команда для отправки локальных коммитов в удалённый репозиторий
  • git pull origin main - команда для получения изменений из удалённого репозитория,объединяя их вашей локальной веткой
