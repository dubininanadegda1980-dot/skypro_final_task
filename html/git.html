Git

  • git status - переводится как проверить состояние, расшифровывается как что сейчас в проекте
  • git add - переводится как добавить файлы,расшифровывается как какие изменения зафиксировать
  • git commit -m -переводится как "Ваше сообщение", расшифровывается как зафиксировать свои изменения локально
  • git push origin main - переводится как отправить в облако, расшифровывается как отправить свои коммиты в облако
  • git pull origin main - переводится как обновить перед новой работой, расшифровывается как получить последние изменения из облака
